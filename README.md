# User and Permission Management in Windows

## Overview
This repository contains My internship work focused on user and permission management
in Windows using Command Prompt.

## Objectives
- Create and manage local users
- Understand group-based access control
- Perform file and folder operations
- Manage NTFS permissions

## Tools Used
- Windows 10
- Command Prompt (Administrator)
- NTFS File System

## Work Summary
User accounts were created using NET USER, group memberships were verified using
NET LOCALGROUP, and file permissions were analyzed using ICACLS.


## Key Concepts
- Least Privilege Principle
- NTFS Access Control Lists
- User Isolation

## Conclusion
This task provided hands-on exposure to real-world Windows system administration tasks.
